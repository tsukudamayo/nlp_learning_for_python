生姜 を 千切り に する 。
小鍋 に ひじき と still たれ 、 酢 を 加え て 炒り煮 し 粗熱 を 取 る 。
★ の 調味 料 の にんにく を すりおろ し た 後 、 他 の ★ の 調味 料 と よく 混ぜ合わせ て フレンチ ドレッシング を 作 る 。
ボウル に キャベツ と ひじき 、 枝豆 、 千切り 生姜 を 入れ て フレンチ ドレッシング と 和え て から 器 に 盛り付け 完成 。
