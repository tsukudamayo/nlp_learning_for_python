みかん は ぬる ま湯 で しっかり と 洗 う 。 上 から 3 分 の 1 ほど の ところ を 横 に 切 り 、 皮 が 割け な い よう に 中身 を と りだ し 果汁 を 搾 る 。
みかん の 寒天 ゼリー の 容器 に 使 う の で 、 中身 を 取り出 し た 皮 は もう 一 度 水 で 洗 い 、 水分 を しっかり と 拭き取 っ て お く 。
鍋 に 水 と 粉 寒天 を 入れ て 混ぜ合わせ 、 砂糖 を 加え て 火 に かけ る 。 沸騰 し た ら その まま 2 分 ほど 混ぜ ながら 煮 る 。
火 を 止め て しぼ っ た 果汁 を 加え て 混ぜ合わせ る 。 果汁 が 冷た い と すぐ に 固ま っ て くる の で 、 素早 く みかん の 皮 の 器 に 注ぎ入れ る 。 冷蔵 庫 で 冷や し て 完成 。
