白ねぎ は 斜め に 切 る 。
鍋 に 白ねぎ 、 水 、 顆粒 だし を 入れ て 火 に かけ る 。 沸騰 し たら ★ の 調味 料 を 入れ 弱火 に し て しばらく 煮 る 。
白ねぎ が 柔らか く な っ た ら 、 牛肉 切り落とし を ほぐ し ながら 加え る 。 全体 の 色 が 変わ っ て 火 が 通 っ たら 、 火 を 止め る 。
うどん は 、 チルド の 場合 は 湯通し し 、 冷凍 の 場合 は 電子 レンジ など で 解凍 する （ パッケージ の 表示 に 従 う ） 。
器 に うどん を 入れ 、 つゆ と 具 を かけ る 。 お 好み で 七味 唐辛子 を ふ っ て 完成 。
