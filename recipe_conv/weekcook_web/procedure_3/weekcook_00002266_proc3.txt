大根 は 、 厚 さ 2 . 5 cm の 輪切り に し て から 皮 を む く 。
大根 を 鍋 に 入れ 、 かぶ る くらい の 水 を 入れ （ あ れ ば 落し蓋 を し て ） 、 強火 で 加熱 する 。 沸騰 し たら 火 を 弱め 、 約 15 分 茹で る 。 茹で上が っ たら 湯 から 出 し て 冷ま し て お く 。 余熱 で 更に 火 が 通 る の で やや 硬め で 火 を 止め る 。
小鍋 に ★ の 調味 料 を 入れ て 混ぜ て から 火 に かけ 、 一 煮立ち し た ら しょう油 を 加え て 火 を 止め て 、 粗熱 を 取 っ て 大根 の 漬け 汁 を 作 る 。
チャック 付き の 保存 袋 （ な けれ ば 保存 容器 など ） を 用意 し 、 大根 を 入れ て から 大根 の 漬け 汁 を 加え 、 保存 袋 内 の 空気 を 抜 い て 口 を 閉じ る 。 大根 が しっかり 浸 る よう に し て 保存 する 。
鍋 に 水 と 0 . 5 % 量 の 塩 （ 分量 外 ） を 入れ て 沸騰 さ せ る （ 1 L の 水 に 塩 小さじ 1 程度 ） 。 沸騰 し たら 小松 菜 を 根元 から 入れ て さっ と 茹で て 、 すぐ に 冷水 に 取 り 冷め た ら 水気 を 絞 り 、 長 さ 5 cm に 切 る 。
大根 の 漬け汁 75 ml と おろ し 生姜 を 鍋 に 入れ て 火 に かけ て 、 沸騰 し たら 鶏 挽肉 を ほぐ し 、 火 が 通 っ たら 火 を 弱め 、 合わせ 味噌 で 味 を 付け る 。 仕上げ に 水溶き 片栗粉 を 回し入れ て 、 とろみ が つ い たら 火 を 止め る 。
大根 は 漬け汁 から 取り出 し 、 電子 レンジ 可能 な 容器 に 入れ て 、 ラップ を し 電子 レンジ （ 500 W ） で 2 分 加熱 し て から 、 大根 と 小松 菜 を 器 に 盛り付け て 、 そぼろ あん を かけ て 完成 。
