鯵 ( 三 枚 おろ し 済 ) は 水 で 洗 い 、 水気 を ペーパー タオル で 拭 く 。
鯵 に 塩 と こしょう を 振 っ て から 薄力 粉 を 全体 に まぶ し 、 溶き卵 を 付け る 。
フライパン を 強火 で 温め て サラダ 油 と バター を 加え た ら 、 鯵 の 身 の 方 を 下 に し て 投入 し 、 2 分 焼 く 。
鯵 を ひっくり返 し て 皮側 を 中火 で 2 分 焼 き フライパン から 取り出 す 。
じゃがいも は 洗 っ て から 水気 が つ い た まま 耐熱 容器 に 入れ 、 電子 レンジ で 4 分 加熱 し て から 皮 を 剥 い て 4 等分 に する 。
鰺 の ピカタ に パセリ を 散ら し て ミニ トマト と じゃがいも 、 レタス を 添え 、 食べ る 直前 に お 好み で ケチャップ を かけ て いただ く 。
