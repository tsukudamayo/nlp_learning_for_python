絹さや は 筋 を 取 っ て から 半分 に 切 る 。
銀杏 は 殻 と 皮 を 剥 い て 茹で る 、 又 は 殻 に 切り込み を 入れ て から 茶 封筒 に 入れ て 電子 レンジ （ 600 W ） で 約 1 分 加熱 し て から 殻 と 皮 を 剥 く 。
ムキエビ は 日本 酒 に 浸け る 。
鶏 もも肉 は 1 cm 角 に 切 っ て から 、 still たれ 少々 に 浸け る 。
卵 を 解きほぐ し て から 、 だし と 白 だし を 加え て よく 混ぜ合わせ て から 一 度 濾 す 。
ムキエビ と 鶏 もも肉 、 銀杏 、 薄切り に し た かまぼこ を 一人 分 ずつ の セット に し て 茶碗 蒸し 茶碗 に 入れ る 。
茶碗 蒸し 用 の 茶碗 に 卵液 を 注 ぎ 、 最後 に 椎茸 と 絹さや を 載せ る 。
