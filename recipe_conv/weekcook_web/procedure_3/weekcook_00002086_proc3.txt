牛乳 は 室温 に 戻 し て お く 。
タラ 切り身 は 皮目 に 切り込み を 入れ 、 塩 と こしょう を 振 る 。 　
じゃがいも は 厚 さ 2 ~ 3 mm の 薄切り に し 、 玉ねぎ 、 にんにく も 薄切り に する 。
じゃがいも と 玉ねぎ 、 にんにく と を 夫々 2 等分 し て グラタン 皿 に 入れ 、 ★ の 調味 料 を 1 / 2 量 ずつ 振りかけ て から 、 ★ の バター を 1 / 2 ずつ し て 各 グラタン 皿 の 材料 の 上 に 載せ て ラップ を し 電子 レンジ で 約 4 分間 加熱 する 。
加熱 し た グラタン 皿 、 夫々 に 牛乳 を 1 / 2 ずつ 入れ 、 タラ の 皮目 を 下 に し て グラタン 皿 の 材料 の 上 に 載せ 、 粉 チーズ を 大さじ 1 ずつ かけ る 。
グリル ( オーブン トースター で も 可 ) また は 、 オープン 250 ℃ で 、 表面 に こんがり 焼き色 が 付 く まで 焼 い たら 完成 。
