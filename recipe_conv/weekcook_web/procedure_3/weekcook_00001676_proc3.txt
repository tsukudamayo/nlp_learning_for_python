田作り は フライパン で じっくり と 弱火 で 乾煎り する 。
少し 冷ま し 、 折 る と ポキッ と 音 が 鳴 る くらい に 仕上げ る 。
一 度 ざる に 取 り 、 田作り の 粉 を 振るい落と す 。
アーモンド スライス は オーブン トースター で こんがり と 焼 く 。 （ 焦げ やす い の で 注意 する ）
フライパン を 綺麗 に し 、 still たれ を 火 に かけ て 沸々 と し て き た ら 田作り と アーモンド スライス 、 胡麻 を 加え て 和え る 。
バット に と っ て 冷ま し 、 器 に 盛り付け 完成 。
