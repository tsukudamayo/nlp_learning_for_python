鶏 もも肉 は 黄色 の 脂肪 を 取り除 き 、 大きめ に 切 る 。
鶏 もも肉 に 塩 、 ブラック ペッパー 、 ハーブ シーズニング を 振 る 。
鍋 に みじん 切り に し た にんにく と オリーブ オイル を 加え て 中火 に かけ 、 香り が 出 る まで 炒め て から 、 鷹 の 爪 を 加え る 。
鍋 に トマト 缶 と トマト の ざく 切 り 、 コンソメ 固形 を 加え て 弱火 で 10 分 煮込 ん で トマト ソース を 作 る 。
フライパン また は グリル で 鶏肉 の 表面 を こんがり と する まで 焼 い て から 、 トマ ソース を 加え て 蓋 を する 。
フライパン の 火 を 中火 に し て 、 灰汁 が 出 たら 取り除 き ながら 柔らか く な る まで 煮込 む 。
チキン の トマト 煮 を 器 に 盛り付け 、 お 好み で パセリ を 散ら し て 完成 。
