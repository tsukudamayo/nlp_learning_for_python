かぼちゃ は 厚 さ 5 mm の 薄切り に し て から 、 長 さ 5 cm に 切 る 。
長 なす は 縦 半分 に 切 っ て から 、 長 さ 5 cm に 切り揃え る 。
長なす を 塩水 ( 分量 外 ) に 約 10 分 浸け て 、 アク抜き する 。
大根 は すりおろ し 、 大葉 は 千切り に する 。
かぼちゃ と 長 なす の 水気 を ペーパー タオル で 拭き取 っ た 後 、 180 度 の サラダ 油 で 素揚げ に する 。
★ の 調味 料 を 合わせ て タレ を 作 り 、 素揚げ に し た かぼちゃ と 長 なす を 器 に 盛り付け て タレ を 振り掛け て から 、 大根 おろし と 大葉 を 載せ たら 完成 。
