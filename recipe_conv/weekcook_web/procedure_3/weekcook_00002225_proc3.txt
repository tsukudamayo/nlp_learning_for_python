春 キャベツ は ざく 切り に する 。 菜の花 は 3 等分 に 切 る 。 ミニ トマト は ヘタ を 取 っ て 半分 に 切 る 。
卵 を ボウル に 割り入れ 溶きほぐ し 、 塩 を 加え る 。
フライパン に サラダ 油 を 入れ て 強め の 中火 で 熱 する 。 卵 を 入れ て ざっと かき混ぜ 、 炒り卵 に し て 取り出 す 。
ペーパー タオル で フライパン を さっと 拭 い た ら 、 マヨネーズ を 入れ て 強火 で 加熱 する 。 野菜 を 全て 入れ て ざっと 炒め 、 塩 を 加え て 一 混ぜ し たら 火 を 止め る 。
野菜 を 器 に 盛り付け 、 炒り卵 を 載せ たら 完成 。
