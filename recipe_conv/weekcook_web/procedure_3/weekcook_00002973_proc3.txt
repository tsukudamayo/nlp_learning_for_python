冷凍 パイ シート （ 18 cm × 18 cm ） は 前日 から 冷蔵 庫 で 解凍 し て お く 。
かぼちゃ は 皮 の 汚れ た 部分 を 除 き 、 2 cm の 角切り に し て 耐熱 容器 に 入れ る 。 ラップ を し て 電子 レンジ （ 600 W ） で 3 分 加熱 し 、 粗熱 を と っ て お く 。
ぶな しめじ は 石づき を 取 っ て ほぐ し て お く 。 玉ねぎ は 薄切り に 、 スライス ベーコン は 1 cm 幅 の 短冊 型 に 切 る 。 プロセス チーズ は 1 cm の 角切り に する 。
ボール に ホワイト ソース を 入れ 、 卵 を 加え て 全体 が 均一 に な る よう に よく 混ぜ合わせ る 。 粉 チーズ ・ 塩 こしょう ・ ナツメグ を 加え て 混ぜ合わせ 卵液 を 作 る 。
フライパン を 温め 、 バター 、 玉ねぎ 、 スライス ベーコン を 入れ て 炒め る 。 玉ねぎ が 透明 に な っ た ら ぶ な しめじ を 加え 、 ぶな しめじ が しんなり し たら 火 を 止め て 粗熱 を と る 。
オーブン を 200 ℃ に 予熱 し て お く 。
パイ シート は 型 より 一 回り 大き く のば し 、 空気 が 入 ら な い よう に 型 に 敷き 、 余 っ た 生地 は カット する 。 フォーク を 使 っ て 底 に 数 カ所 穴 を 開け る 。
パイ シート の 上 に 卵液 の 半量 を 流し入れ 、 かぼちゃ と チーズ 、 玉ねぎ 、 スライス ベーコン 、 ぶな しめじ を 並べ 、 残り の 卵液 を 流 し 、 予熱 し た オーブン で 20 ～ 25 分 、 こんがり と 焼き目 が つ く まで 焼 き 、 取り出 し たら 完成 。
