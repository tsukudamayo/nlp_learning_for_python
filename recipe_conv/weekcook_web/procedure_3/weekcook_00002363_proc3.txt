さや いんげん は さっ と 茹で て から 半分 に 切 る 。
鶏肉 に 溶き卵 を 加え て 混ぜ 、 片栗粉 を 加え て 170 度 の サラダ 油 で 揚げ 、 最後 は 180 度 に 上げ て カリッ と 揚げ る 。

鶏 の から揚げ と さや いんげん の 甘辛 を 器 に 盛り付け 、 いり 胡麻 を 振 っ たら 完成 。
