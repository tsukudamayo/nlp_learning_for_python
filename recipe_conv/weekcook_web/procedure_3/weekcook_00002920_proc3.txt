ブリ は 余分 な 水分 を ペーパー タオル で 拭き取 っ て から 日本 酒 を 振 る 。

ブリ を 焼 き 終え た フライパン を 綺麗 に し て から 、 再び 火 に かけ 、 サラダ 油 ( 大さじ 1 ) を 加え て 玉ねぎ と ピーマン を 炒め る 。
玉ねぎ が しんなり し た ら 、 塩 と こしょう 、 カレー 粉 で 味 を 付け ★ の 調味 料 を 加え たら 火 を 止め る 。
焼け た ブリ を お 皿 に 盛り付け 、 ブリ の 上 に 炒め た 野菜 を 乗せ たら 完成 。
