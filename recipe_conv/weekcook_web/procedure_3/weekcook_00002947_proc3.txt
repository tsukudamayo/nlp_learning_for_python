厚揚げ と エリンギ は 食べ 易 い 大き さ に 切 る 。
さや いんげん は へた を 落と し て 水 で 洗 い 、 電子 レンジ （ 600 W ） で 約 1 分 加熱 する 。
加熱 し た さや いんげん を 冷水 に 浸け た 後 、 水気 を 取 っ て から 半分 に 切 る 。
フライパン で 厚揚げ を 片面 1 分 ずつ の 素焼き に し 、 厚揚げ を 取り出 し た 後 、 エリンギ を 約 1 分 の 素焼き に する 。
フライパン に 厚揚げ を 戻し入れ 、 水 と ★ の 調味 料 を 加え て から 火 に 約 5 分 かけ る 。
更に 、 さや いんげん と かつお節 を 加え た 後 、 強火 で 30 秒 加熱 し て 味 を なじ ま せ たら 火 を 止め る 。
厚揚げ と エリンギ の おかか 煮 を 器 に 盛り付け 完成 。
