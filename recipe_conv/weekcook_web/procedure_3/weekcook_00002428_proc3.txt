あさり （ 砂抜き 済み ） は 、 殻 同士 を 擦り合わせ る よう に し て 流水 で 洗 う 。
筍 （ 水煮 ） は 厚 さ 3 〜 4 mm に 切 る 。 （ 穂先 側 が お 薦め ・ 筍 を 茹で る 場合 は ポイント を 参照 ） 新 玉ねぎ は 繊維 と 直角 に 厚 さ 5 mm に 切 る 。 えのき茸 は 根元 を 切り落と し 、 長 さ 半分 に 切り ほぐ す 。 にんにく は 薄切り に する 。
半分 に 折 っ た パスタ が 入 る 大き さ の 耐熱 容器 等 に パスタ を 入れ て 、 水 500 ml と ★ の 調味 料 を 加え る 。 ラップ を せ ず に 電子 レンジ で 加熱 する 。 加熱 時間 は 袋 に 記載 さ れ た 茹で 時間 ＋ 2 分 半 。 （ 500 W の 場合 ）
フライパン に オリーブ オイル と にんにく を 入れ て 熱 し 、 細か い 気泡 が 出 て き た ら 筍 と 新 玉ねぎ を 入れ て さっ と 炒め 、 塩 を 振 る 。 あさり と えの き 、 日本 酒 を 加え て 蓋 を し 、 3 ～ 5 分 あさり の 殻 が 開 く まで 蒸し焼き に する 。
パスタ の 加熱 が 終わ っ た ら 、 ざる に あげ て 水気 を 切 り 、 フライパン に 加え る 。 30 秒 ほど 加熱 し ながら ソース と パスタ を 絡め る 。
お 皿 に 盛り付け て 、 お 好み で パセリ を 散ら し たら 完成 。
