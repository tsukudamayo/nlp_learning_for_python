玉ねぎ は みじん 切り に し て から 、 バター で 2 ～ 3 分 炒め て 冷ま し て お く 。
ボウル に 合挽肉 と 玉ねぎ の みじん 切り 、 ★ の 食材 と 調味 料 を 入れ 、 手 で 混ぜ合わせ 粘り が 出 て き た ら 4 等分 に し て 、 手 に 打ちつけ て 空気 を 抜 き ながら 楕 円 形 に 形づくり 、 中央 を 少し へこ ま せ ハンバーグ の タネ を 作 る 。
フライパン で サラダ 油 を 熱 し 、 ハンバーグ の タネ を 入れ て 中火 で 1 分 ほど 焼 き 、 焼き色 が つ い たら 裏返 し て 弱火 に し 、 蓋 を し て から 6 ～ 7 分 焼 い た 後 に 取り出 す 。
ブロッコリー は 小房 に 分け て から 加熱 用 の 器 に 入れ 、 ラップ を し て 電子 レンジ （ 600 W ） で 約 1 分 加熱 する 。
電子 レンジ 可能 な 容器 で Ａ の 材料 を 混ぜ合わせ て ラップ を し 、 電子 レンジ （ 600 W ） で 約 2 分 加熱 し て 、 一 度 取り出 し て 混ぜ た 後 、 更に 約 2 分 加熱 し た ホワイト ソース に チーズ （ ピザ 用 ） を 加え る 。
耐熱 皿 に ハンバーグ と ブロッコリー 、 コーン 缶 （ 粒 ） を 入れ 、 チーズ 入り の ホワイト ソース を かけ る 。
オーブン トースター 等 で 焼き色 が つ く まで 焼 い て 完成 。
