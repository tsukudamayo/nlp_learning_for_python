レンコン は 皮 を む い て から 、 厚 さ 3 ～ 5 mm の 輪切り に する 。
フライパン に ★ の 調味 料 と 水 25 ml （ 分量 外 ） を 入れ て 加熱 し 、 沸騰 し たら レンコン を 加え る 。 1 分 半 ～ 2 分 茹で たら 火 を 止め 、 その まま 粗熱 を 取 っ て 漬け て お く 。
セロリ は 筋 を と っ て 、 2 mm 角 程度 の 細 い 千切り に する 。
ボウル に 、 酢蓮 と オリーブ オイル を 入れ て 軽 く 和え て から 、 セロリ と 塩 を 加え て 更に 混ぜ合わせ る 。 （ 漬け汁 は 加え な い ） 器 に 盛り付け て 、 粗挽き 黒 こしょう を 振 っ て 完成 。
