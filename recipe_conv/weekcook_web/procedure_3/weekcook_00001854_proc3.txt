ごぼう は 皮 を こす り 落と し て から 長 さ 1 cm 切 り 、 酢水 ( 分量 外 ) に 、 さっと 、 さら し て から 、 ざる に 上げ る 。
ごぼう は 水 から 約 2 分 茹で る
豚肩 ロース ( ブロック ) は 、 一口 大 に 切 る 。
鍋 を 火 に かけ 、 サラダ 油 ( 分量 外 ) を 注 い で から 豚肉 を 焼 い て 焼き色 を 付け た ら 、 ★ の 水 と 調味 料 、 及び 大豆 水煮 、 茹で た ごぼう を 加え て 、 豚肉 が 柔らか く な る まで 弱火 で 約 20 分 煮 る 。
大豆 と 豚肩 ロース の 味噌 煮 を 器 に 盛り付け たら 完成 。
