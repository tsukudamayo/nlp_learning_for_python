真鯛 （ 切身 ） に 塩 と こしょう を し て お く 。
砂抜き し た あさり は 、 殻 同士 を こすり合わせ て 洗 っ て お く 。
にんにく は 半分 に 切 っ て 真ん中 の 青 い 部分 を 除 い て お き 、 ミニ トマト は 洗 っ て ヘタ を 取 る 。
フライパン に オリーブ オイル と にんにく を 入れ て 火 に かけ 、 香り が 立 っ た ら 真鯛 を 並べ て 両面 に 焼き色 を 付け る 。
真鯛 に 焼き色 が 付 い た ら あさり と ミニ トマト 、 オリーブ 塩漬け を 加え 、 水 、 白 ワイン を 回しかけ て 蓋 を し 、 あさり の 殻 が 開 く まで 蒸し煮 に する 。
真鯛 の アクアパッツァ を 器 に 盛り付け 、 仕上げ に イタリアン パセリ の 刻 ん だ もの を 散ら し て 完成 。
