豚肉 は 脂肪 と 赤身 の 間 に 数 カ所 切り込み を 入れ て 肉 の 筋 を 切 っ て お く 。 両面 に 塩 と 胡椒 を ふ っ て 下味 を つけ 、 片栗粉 を 薄 く まぶ す 。
舞茸 は 適当 な 大き さ に ほぐ す 。
パプリカ は 5 mm 幅 に 切 る 。
フライパン に サラダ 油 を 入れ て 中火 で 熱 し 、 油 が 温ま っ て き た ら 回 し 広げ 、 豚肉 を 加え て 、 焼き色 が つ く まで １ 〜 ２ 分 焼 く 。
裏返 し 、 フライパン の あい て い る 部分 に 舞茸 と パプリカ を 入れ て 蓋 を する 。 弱火 で １ 〜 ２ 分 焼 く 。
蓋 を 開け て 火 を 強め 、 しょう油 と みりん を 加え 、 煮詰め ながら 絡め る 。
お 皿 に 盛り付け て ブラック ペッパー を ふ っ て できあがり 。
