長芋 は 皮 を む い て 3 等分 に 切り分け 、 厚 さ 1 cm に 切 る 。 水菜 は 洗 っ て 長 さ 3 cm に 切 り 、 水分 を 拭 う 。
豚 もも肉 薄切り を 広げ て 塩 と こしょう を 振 る 。 豚肉 を 長芋 に 巻きつけ て から 薄力 粉 を まぶ す 。
フライパン を 温め て サラダ 油 を 加え 、 長芋 の 肉巻 を 約 1 分 焼 い て 裏返 し 、 弱火 に し て 約 3 分 、 転が し ながら 全面 に 焼き色 が 付 く まで 焼 く 。 フライパン を 火 から 離 し 、 still たれ を 加え て 長芋 の 肉巻 に 絡め る 。
水菜 を 敷 い た 器 に 長芋 の 肉巻 を 盛り付け 完成 。
