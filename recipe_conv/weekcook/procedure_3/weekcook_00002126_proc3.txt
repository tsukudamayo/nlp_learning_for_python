ムキエビ は 塩水 （ 分量 外 ） で さっ と 洗 い （ 背わた が 気 に な る 様 なら 取 る ） 、 水気 を 拭き取 っ て から ★ の 調味 料 を 揉み込 ん で 下味 を つけ る 。
カリフラワー は 小房 に 分け 、 にんにく は 薄切り に する 。
鍋 に 湯 を 沸か し 、 カリフラワー を 1 ～ 2 分 茹で て 取り出 し た 後 、 続 い て エビ を 投入 し て 色 が 変わ る まで さっ と 茹で た ら 取り出 し て お く 。
ボウル で 塩 と Ａ の 調味 料 を 混ぜ合わせ て マヨネーズ ソース を 作 り 、 別 の ボウル に 卵 を 割りほぐ し て お く 。
中華 鍋 （ フライパン で も 可 ） を 熱 し て サラダ 油 大さじ 1 を 注ぎ入れ 、 溶き卵 を 加え て 炒り卵 を 作 っ た ら 取り出 す 。
残り の サラダ 油 を 中華 鍋 に 足 し て から にんにく を 炒め 、 香り が 出 たら エビ と カリフラワー を 加え て 炒め た 後 、 炒り卵 を 戻し入れ る 。
