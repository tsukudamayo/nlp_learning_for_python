セロリ の 葉 の 部分 （ 分量 外 ） を 3 枚 ほど 千切り に する 。
鶏 む ね 肉 に 塩 と こしょう を まぶ し 、 千切り に し た セロリ の 葉 、 日本 酒 と とも に チャック 付き ポリ袋 に 入れ 、 軽 く もん だ ら 空気 を 抜 い て 袋 を とじ 、 漬け込 む 。
生姜 は 千切り 、 セロリ は 5 mm 厚 さ の 輪切り に する 。
電子 レンジ 対応 の お 皿 に 鶏 む ね 肉 と セロリ を 盛り付け 、 つけ 汁 と 生姜 を ふりかけ る 。 ラップ を し て 、 電子 レンジ （ 600 W ） で 3 分 間 加熱 する 。 その まま 5 分 ほど 置 い て お き 、 余熱 で 火 を 通 す 。
トマト は 7 mm 角 に 切 る 。
トマト に 塩 と 胡麻 油 を 加え て 混ぜ合わせ 、 ソース を つく る 。
火 を 通 し た 鶏肉 を 食べ やす い 大き さ に 切 り 、 お 皿 に 盛り付け て トマト ソース を かけ た ら 、 完成 。
