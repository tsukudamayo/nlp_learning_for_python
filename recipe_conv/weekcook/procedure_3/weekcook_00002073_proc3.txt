海老 芋 に つ い た 泥 を 流水 で 洗い落と し て 、 海老 芋 が かぶ る くらい の 塩水 （ 分量 外 ） で 、 水 から 茹で る （ 竹串 を 刺 し 、 すっ と 通 れ ば 柔らか く な っ て い る ） 。
お 湯 から 上げ 、 水 に さら し た 後 、 皮 を 剥 く 。
海老 芋 を 拍子 木 切 り 、 金時 人参 と ベーコン は 千切り に する 。
オーブン を 200 ℃ に 予熱 し て から 、 タルト 型 また は グラタン 皿 に クッキング ペーパー を 丸 く 切 っ て 敷 く 。
ボウル に 卵 を 溶きほぐ し 、 生 クリーム を 加え 、 泡立て 器 で よく 混ぜ合わせ て から 、 塩 と こしょう で 調味 する 。
そこ へ 海老 芋 と 金 時 人参 、 ベーコン を 加え て 混ぜ合わせ キッシュ の 素 を 作 る 。
クッキング ペーパー を 敷 い た タルト 型 に キッシュ の 素 を 入れ 、 縁 まで しっかり 詰め て 、 表面 に チーズ を 散ら す 。
オーブン に タルト 型 を 入れ 、 加熱 （ 約 25 分 ） し たら 完成 。
