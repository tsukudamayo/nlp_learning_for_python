あさり ( 砂抜き 済み ) は 貝 同士 を こすりあわせ て 洗 う 。
レモン は くし 型 に 切 っ て から 横 半分 に 切 り 、 にんにく は みじん 切り に する 。
玉ねぎ と パプリカ 赤 、 パプリカ 黄 は 2 cm 角 に 切 る 。
サフラン は 大さじ 1 の 水 ( 分量 外 ) に 浸け る 。
フライパン を 火 に かけ 、 オリーブ オイル を 注ぎ入れ て 温め て から 、 カット し た 玉ねぎ と にんにく を 炒め 、 米 は 洗 わ な い で その まま 加え て 更に 炒め る 。
全体 に 油 が 馴染 ん だら 、 トマト 缶 の 内容 物 と 水 を 合わせ て 300 ml に し て から フライパン に 注ぎ入れ 、 サフラン と コンソメ 固形 を 加え た 後 、 塩 と こしょう で 味 を 調え る 。
煮汁 が 沸々 と し て き た ら 、 シーフード ミックス と あさり を 米 の 上 に 載せ て 白 ワイン を 振 っ たら 蓋 を し て 強火 に し た 後 、 沸騰 し て き たら 弱火 に し て 約 10 分 加熱 し 、 火 を 止め て 約 5 分 蒸ら す 。
