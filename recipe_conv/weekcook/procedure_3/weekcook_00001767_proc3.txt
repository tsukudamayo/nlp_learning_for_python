生鮭 は 余分 な 水分 を ペーパー タオル で ふき取 り 酒 （ 分量 外 ） を 振 る 。
ごぼう は 斜め に 約 2 ~ 3 mm の 厚 さ に 切 り 、 水 （ 分量 外 ） に つけ て 、 アク抜き を する 。
ぶな しめじ は 石突き を 取り ほぐ す 。
赤 唐辛子 を 輪切り 、 にんにく を 薄切り に する 。
深め の 耐熱 ボール に 、 水気 を 切 っ た ごぼう と Ａ の 調味 料 を 入れ ラップ を し て レンジ ( 800 W で 約 4 分 ) にて 加熱 する 。
加熱 し た ごぼう に 、 ぶな しめじ と コンソメ 顆粒 を 加え て 混ぜ合わ せ 、 塩 で 味付け 後 ラップ を し て レンジ （ 800 W ） で 約 4 分 加熱 する 。
加熱 し た ペペロンチーノ 風 付け あわせ に 、 Ｂ の 調味 料 を 加え よく 混ぜ合わせ る 。
★ の 調味 料 を 合わせよく 混ぜ る 。
生鮭 の 皮目 を 下 に し て グリル ( フライパン で も 可 ) に 並べ 、 表面 に ★ の 調味 料 を 塗り 約 10 分 加熱 する 。
器 に さけ の マヨ 焼き を 盛り付け て 、 ペペロンチーノ 風 付け あわせ を 添え て 完成 。
