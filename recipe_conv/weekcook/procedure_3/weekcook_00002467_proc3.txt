鯵 ( 二 枚 おろし の 物 ) は 水分 を 拭き取 り 、 塩 を 振 っ て 10 分 寝かせ る 。
新 玉ねぎ は 皮 を 剥 い て 薄切り 、 セロリ も 薄切り 、 人参 は 皮 を 剥 い て から 千切り に する 。
新 玉ねぎ は 水 に さら し て から 水気 を ペーパー で ふき取 る 。
鯵 から 出 て き た 水分 を 拭き取 り 、 塩 、 こしょう を 振り 片栗粉 を 全体 に つけ る 。
片栗粉 を 付け た 鯵 を 、 180 ℃ の サラダ 油 で からり と 揚げ る 。
鷹 の 爪 は 小口 切り に し て から 、 ★ の 調味 料 と 合わせ て 一 煮立ち さ せ 、 たれ を 作 る 。
蓋付 の 容器 に 野菜 と 揚げ た 鯵 、 たれ を 入れ て 浸け る 。
