牛 ロース 肉塊 に 塩 と ブラック ペッパー を 振 り 、 おろし にんにく を すり込 む 。
フライパン を 中火 強 で 温め て 牛脂 （ 分量 外 ） を 鍋肌 に 擦り付け て から 、 牛 ロース 肉 の 各面 を 約 1 分 ずつ 焼 く 。
焼 い た 牛 ロース 肉 は ラップ で 包 み 、 チャック 付き 保存 用 袋 に 入れ て 空気 を 抜 い た 後 、 チャック を ぴったり と 閉じ る 。
厚手 の 鍋 に 湯 2 L （ 分量 外 ） を 注 ぎ 、 チャック 付き 保存 用 袋 の まま の 肉 を 入れ て 80 ℃ に 保 ち ながら 20 分 ～ 50 分 保温 する 。
鍋 から 肉 の 入 っ た チャック 付き の 保存 用 袋 を 取り出 し 、 肉 の 粗熱 が 取れ たら ラップ を 外 し て 、 肉 と still だ し たれ を チャック 付き の 保存 用 袋 に 加え る 。
ロースト ビーフ を 浸け て い た still だ し た れ は 少し 煮詰め て から 冷ま し て お く 。
