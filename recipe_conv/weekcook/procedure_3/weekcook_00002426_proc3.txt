玉ねぎ と 新 じゃがいも 、 人参 は 食べ 易 い 大き さ に 切 る 。 ブロッコリー は 小房 に 分け て 、 ミニ トマト は スライス する 。 卵 は 常温 に 戻 し て お く 。
鍋 に サラダ 油 を 熱 し 、 牛角 切り 肉 を 炒め る 。 肉 の 表面 に 焼き色 が 付 い た ら カレー 用 野菜 を 加え て 炒め る 。
鍋 に 水 を 加え 、 煮立 っ たら 中火 に し て 約 20 分 煮 る 。 一旦 火 を 止め て カレー ルー を 加え る 。 混ぜ合わせ て ルー が 溶け たら 火 を つけ て 約 5 分 煮 る 。
鍋 に 卵 を 入れ 、 かぶ る くらい の 水 を 加え て 火 に かけ る 。 沸騰 し たら 少し 火 を 弱め 、 沸騰 し て から 約 10 分 茹で る 。 火 を 止め て その まま しばらく 放置 し て 触れ る 程度 に 冷め た ら 冷水 に 取 り 、 殻 を 剥 く 。
茹で卵 の ウサギ を 作 る 。 飾 る とき に 底 に な る 白身 の 部分 を 切り取 る 。 それ を 半分 に 切 っ て 耳 に する 。 頭頂部 に あた る 部分 に 切り込み を 入れ て 耳 を 差し込 む 。 のり など で 目 や 鼻 、 口 を かたど っ て 切 り 、 ウサギ の 顔 を 作 る 。
ブロッコリー は 電子 レンジ で 下茹で する 。
お 皿 に ごはん を 盛 り 、 茹で卵 の ウサギ を 飾り カレー を 注 ぎ 、 ブロッコリー と ミニ トマト を 添え た ら 、 完成 。
