カジキマグロ は 水分 を ペーパー タオル で 拭き取 っ て から 日本 酒 を 振 る 。
カジキマグロ は 1 cm 幅 に 切 り 、 塩 と こしょう を 振 っ て から 薄力 粉 を 薄 く まぶ す 。
さや いんげん は 電子 レンジ で 1 分 程度 加熱 し て から 冷水 に 取 っ た 後 、 水分 を ペーパー タオル 等 で 拭 っ て から 半分 に 切 る 。
玉ねぎ は 1 / 8 個 分 を 生姜 と 一緒 に すりおろ し 、 残り 3 / 8 個 分 は 櫛切り に する 。
フライパン に サラダ 油 を ひ い て 中火 強 で 櫛切り の 玉ねぎ を 炒め て から 取り出 し 、 次 に カジキマグロ を フライパン で 焼 く ( 2 分 程度 ) 。
★ の 調味 料 と すりおろ し た 玉ねぎ 、 生姜 を 混ぜ合わせ て ピリ辛 たれ を 作 る 。
2 分 程 焼 い て カジキマグロ の 周り の 色 が 変わ っ て き たら ひっくり返 し 、 更に 1 ～ 2 分 焼 く 。
玉ねぎ と さや いんげん を 戻し入れ た ら 、 フライパン を 火 から 離 し て ピリ辛 たれ を 加え 、 再び 火 に かけ て 絡め る 。
カジキマグロ の ピリ辛 炒め を お 皿 に 盛り付け て 完成 。
