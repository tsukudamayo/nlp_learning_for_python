きゅうり は 千切り に し 、 ★ の 生姜 は みじん 切り に する 。
小鍋 に ★ の 生姜 の みじん 切り と ★ の 調味 料 を 入れ て 火 に かけ 、 煮詰め て 甘味噌 を 作 る 。 ( でき た 甘味噌 は 別 の 容器 に 移 し て お く )
小鍋 で 、 鶏 ミンチ と 日本 酒 大さじ 1 を かき混ぜ て から 火 に かけ て 炒り煮 する 。
鶏 ミンチ が ほぐれ た ら 、 容器 に 移 し た 甘味噌 で 味 を 付け 、 仕上げ に 豆板 醤 と ラー油 を 加え て から 火 を 消 す 。 ( 肉 味噌 )
中華 麺 は たっぷり の お 湯 で 、 包装 袋 に 記載 さ れ た 作 り 方 通り に 茹で た 後 、 冷水 に 取 っ て 水気 を しっかり 切 っ た ら 胡麻 油 少々 ( 分量 外 ) を 回し掛け る 。
器 に 麺 を 盛 っ て から 、 その 上 に きゅうり 、 最後 に 肉 味噌 を 盛り付け て 完成 。
