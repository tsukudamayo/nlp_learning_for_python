大根 は 3 ～ 4 ㎝ の 厚 さ に 切 り 、 厚め に 皮 を む い て 面取り する 。 底面 に 十字 に 隠し 包丁 を 入れ る 。
鍋 に 面取り し た 大根 を 並べ 、 米 の とぎ汁 （ な けれ ば 米 ひと つかみ と 水 ） を 大根 が かぶ る 程度 に 入れ て 火 に かけ る 。 煮立 っ たら 中火 に し て 15 分 ほど 煮 る 。
大根 を 取り出 し 、 鍋 に 昆布 を 敷 い て その 上 に 大根 を 並べ る 。 大根 が かぶ る 程度 に 水 を 入れ て 火 に かけ 、 煮立 っ たら 中火 に し て 20 ～ 25 分 ほど 煮 る （ 竹串 が スッ と 通 る 程度 まで 煮 る ） 。
柚子 は 半分 に 切 っ て 搾 り 、 大さじ 1 の 分量 を 取 っ て お く 。 皮 は 飾り 用 に 刻 ん で お く 。
鍋 に Ａ の 柚子 味噌 の 材料 と 柚子 の 搾り 汁 （ 大さじ 1 ） を 入れ 、 味噌 を 溶 く 。 中火 に かけ て 焦が さ な い よう に 混ぜ 、 とろり と する まで 煮詰め たら 火 から 下ろ す 。 同様 に 別 の 鍋 に Ｂ の 胡麻 味噌 の 材料 を 全て 入れ 胡麻 味噌 を 作 る 。
