木綿 豆腐 は 縦 半分 に 切 っ て から 厚 さ 1 cm に 切 り 、 耐熱 皿 に ペーパー タオル 2 枚 を 敷 い た 上 に 切 っ た 木綿 豆腐 を 載せ る 。 ラップ なし で 、 電子 レンジ ( 500 W ) で 約 3 分 加熱 する 。
ニラ は 長 さ 3 cm に 切 り 、 生姜 は すりおろ し て 汁 を 絞 っ て お く 。
ホタテ 水煮 缶詰 ( 小 ) の 内容 物 を 汁 ごと ボウル に 入れ て から ホタテ を ほぐ す 。 ★ の 水 と 調味 料 を 加え て 混ぜ合わせ る 。
フライパン に 胡麻 油 を 注 い で 強火 に かけ 、 豆腐 を 加え て 両面 を 焼 く 。 良 い 焼き色 が 付 い た ら 、 ニラ と ボウル の ホタテ 及び 汁 を 加え て 一 煮立ち さ せ る 。 とろみ が つ い たら 火 を 止め て から 生姜 汁 を 加え て 混ぜ る 。
