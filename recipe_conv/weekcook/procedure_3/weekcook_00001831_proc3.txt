玉ねぎ は みじん 切り に する 。
パン 粉 は 牛乳 に 浸 す 。
ボウル に 、 合挽肉 → 玉ねぎ → 水分 を 絞 っ た パン 粉 → 卵 → 塩 、 こしょう を 順 に 入れ たら 、 よく 練り混ぜ て 肉詰め の タネ を 作 る 。
ピーマン は 縦 半分 に 切 っ て 種 を 取り除 き 、 切 っ た ピーマン に 肉詰め の タネ を こんもり と 盛り付け 、 タネ に パン 粉 （ 分量 外 ） を まぶ す 。
フライパン に サラダ 油 を 入れ て 火 に かけ 、 ピーマン の 肉詰め を タネ 側 を 下 に し て フライパン に のせ 、 弱火 で 約 3 分 焼 く 。
裏返 し て 蓋 を し た 後 、 約 2 分 焼 き 、 中 まで 火 が 通 っ たら 取り出 し た 後 、 ケチャップ と ウスター ソース を 混ぜ合わせ て から フライパン に 入れ て ソース を 作 る 。
