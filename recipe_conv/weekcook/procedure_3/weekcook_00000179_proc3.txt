豚 ロース （ ステーキ 用 ） は 筋切り を し て から 、 塩 と こしょう を 振 る 。
筋 切り し た 豚肉 に 軽 く 薄力 粉 を まぶ す 。
玉ねぎ は すりおろ し 、 他 の ★ の 調味 料 と 混ぜ て ポーク ステーキ 用 の ソース を 作 る 。
フライパン を 温め て から 、 サラダ 油 を 加え て 豚肉 を 焼 く 。
豚肉 の 周り に 出 て き た 油 を キッチン ペーパー で 拭き取 っ て から 、 ソース を 加え て 絡め る 。
お 皿 に ポーク ステーキ を 盛り付け て 、 飾り に ベビー リーフ を その 脇 に 盛り付け たら 完成 。
