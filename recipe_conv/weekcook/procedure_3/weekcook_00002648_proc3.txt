大葉 は 千切り に し て 水 に さら し た 後 、 ペーパー タオル で 水気 を 拭 う 。
鍋 に 水 を 入れ て 火 に かけ 、 沸騰 し たら 塩 を 入れ て から パスタ を 加え て パッケージ の 表示 通り に 茹で る 。 ( パスタ の 茹で 汁 100 ml は 取 っ て お く )
トッピング 用 に かね ふく 辛子 明太子 （ 切子 ） を 長 さ 2 cm 分程 残 し て 、 残り は 薄皮 を 切り開 い て 中身 を 取り出 し 、 昆布 茶 と 一緒 に ボウル に 入れ て 置 く 。
パスタ が 茹で上が っ た ら 、 明太 子 と 昆布 茶 の 入 っ た ボウル に 茹で 汁 ( 100 ml ） を 加え て 昆布 茶 を 溶か し て から パスタ を 入れ て 和え る 。
