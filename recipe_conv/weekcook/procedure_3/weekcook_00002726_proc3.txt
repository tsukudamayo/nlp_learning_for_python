パスタ が 入 る 大き さ の 容器 や タッパー に 、 半分 に 折 っ た パスタ （ 折 ら ず に 入 り そう なら 折 ら な く て も 良 い ） を 入れ 、 水 500 ml 、 塩 小さじ 1 / 2 、 オリーブ オイル 小さじ 1 を 加え る 。
ラップ を せ ず に 電子 レンジ （ 500 W ） で 加熱 する （ 500 W の 場合 、 加熱 時間 は パスタ の 袋 に 記載 さ れ た 茹で 時間 ＋ 2 分 半 ） 。
ぶな しめじ は 石づき を と っ て 軽 く ほぐ し て お く 。 にんにく は 薄切り に し 、 鷹 の 爪 は ヘタ を 切り落と し 種 を 取り除 く 。 ベ ビー チーズ は 角切り に し 、 トマト は 厚 さ 1 cm 程度 の くし切り に する 。
フライパン に オリーブ オイル 、 にんにく 、 鷹 の 爪 を 入れ て 弱火 に かけ る 。 にんにく から 細か い 気泡 が 出 て き た ら 、 ぶな しめじ を 加え て 炒め る 。
ぶな しめじ が しんなり と し て き た ら 、 鯖 水煮 缶 の さば の 身 を 加え て 一口 大 に ほぐ し ながら 軽 く 炒め て いったん 火 を 止め る 。
パスタ が ゆで上が る 少し 前 に 、 再度 中火 に かけ 、 トマト 、 チーズ を 加え て さっ と 炒め る 。 鯖 水煮 缶 の 缶汁 と パスタ の ゆで汁 大さじ 1 も 加え て 揺す り ながら 混ぜ る 。
ゆであが り 30 秒 前 の パスタ を 加え 、 かきまぜ ながら パスタ と ソース を なじ ま せ て 完成 。
