ブロッコリー を 小房 に 分け る 。 お 湯 に 1 % 程度 の 食塩 （ 分量 外 ） を 加え 沸騰 さ せ 、 ブロッコリー を 約 2 分 茹で る 。 茹で上が っ た ら ザル に あげ る 。
エリンギ は 食べ 易 い 大き さ に 裂 く 。 レンコン は 皮 を む い て 厚 さ 7 mm の 半月 切り に する 。 玉ねぎ は 櫛切 り 、 セロリ は 厚 さ 3 mm の 輪切り 、 トマト は ざく 切り に する 。 切 っ た もの から 鍋 に 入れ る 。
タラ 切り身 の 両面 に 、 塩 （ 分量 外 ） と こしょう （ 分量 外 ） を 振 っ て お く 。 タラ 切り身 は ３ 等分 に 切 る 。
鍋 に コンソメ 顆粒 、 水 を 入れ て 7 ～ 9 分 加熱 する 。
玉ねぎ に 火 が 通 っ て 透き通 っ たら 、 ケチャップ と みりん 、 おろし にんにく を 加え る 。 更に タラ と トマト を 加え 、 ブロッコリー は 蕾 部分 が 煮汁 に つか ら な い よう に し て 加え 、 蓋 を し て 更に 約 3 分 煮る 。
具材 に 火 が 通 っ たら 、 お 好み で 粗挽き 黒 こしょう や 粉 チーズ 、 粒 マスタード など を 加え たら 完成 。
