鰯 ( 頭 と 腹 を 取 っ た 物 ) は ペーパー タオル で 水気 を 拭き取 る 。
圧力 鍋 に 鰯 と 酢 、 still た れ 、 ザラメ を 入れ て 蓋 を 閉め て から 火 に かけ 、 リング が 上が っ て から 約 15 分 加圧 する 。
リング が 下が っ た ら ( 減圧 ) 、 粉末 に し た かつお節 と すり 胡麻 大さじ 3 を 加え て 水分 が 無くな る まで 火 に かけ て から 、 火 を 止め る 。
容器 に 残り の すり 胡麻 半量 を 広げ て 鰯 を 並べ 、 並べ た 鰯 の 上 に も すり 胡麻 半量 を 塗 す 。
