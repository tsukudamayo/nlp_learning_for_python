合挽肉 は チャック 付き の ポリ袋 に 入れ 、 空気 を 入れ て チャック を 閉じ 、 冷蔵 し て お く 。 ※ 調味 料 を 加え て こね る の に 使 う の で 、 ひき肉 の 量 に 対 し て 少し 大き 目 の もの が 良 い 。
ベビー チーズ は 7 mm 角 程度 の さい の 目 に 切 る 。
新 じゃがいも は 皮 を よく 洗 い 、 1 cm の 厚 さ に 切 っ て 水 に 浸 し て お く 。
玉ねぎ は 皮 を む い て 先端 と お 尻 を 切り落と し 、 5 mm 程 の 薄切り 、 パプリカ は 長 さ を 半分 に 切 っ て 拍子 切 り 、 舞茸 は 適当 な 大き さ に さ く 。
合挽肉 が 入 っ た ポリ袋 に 、 塩 、 片栗粉 、 水 、 こしょう を 加え 、 袋 の 外 から もん で よく こね る 。 ある 程度 粘り が 出 て き たら 、 チーズ も 加え て さらに よく 混ぜ る 。
袋 から 肉 だね を 取り出 し 、 5 等分 し て 丸め て 肉 団子 に する 。 ※ ひき肉 は 冷た い うち に 丸め る と 手 に ひき肉 や 脂 が つ き にく い 。 チーズ が 表面 に 出 な い よう に する と 、 煮込 ん だ 時 に チーズ の 流出 が 抑え られ る
フライパン に オリーブ オイル を 入れ て 中火 で よく 熱 し 、 丸め た 肉 だ ね と 水気 を 切 っ た 新 じゃがいも を 入れ る 。 全体 に 焼き色 が つ く よう に 動か し ながら 4 分 程度 焼 き 、 肉 団子 に 火 が 通 っ たら 取り出 す 。
