鍋 に 水 1 リットル ( 分量 外 ) と 塩 を 入れ て 湯 を 沸か す 。
玉ねぎ は 薄切り に し て 、 小松菜 は 根元 を 切り落と し 、 4 cm 幅 に 切 っ て から 、 茎 と 葉 に 分け て お く 。
ミニ トマト は へた を と っ て 4 等分 に 切 り 、 にんにく は 薄切り に し て 、 ツナ 缶 ( 小 ) の 蓋 を あけ て お く 。 （ ※ 汁 は 捨て な い ）
湯 が 沸 い た ら パスタ を 加え 、 再度 沸騰 し たら ふつふつ と する 程度 に 火力 を 調節 し て 、 パスタ 同士 が くっつ か な い よう に 軽 く 混ぜ る 。 ( 袋 の 指示 より 1 分 短め に 茹で る )
この 後 、 コンロ が 2 口 あ る 場合 は 茹で て い る 間 に パスタ ソース を つく る 。 ( 1 口 の 場合 は 、 先 に ソース を 作 っ て お く → 麺 を ゆで る → ゆであが っ た 麺 を パスタ ソース に 加え る )
フライパン に オリーブ オイル と にんにく を 入れ て 弱火 で じっくり 加熱 し て 、 にんにく が うっすら 色づ い て き た ら 玉ねぎ と ミニ トマト を 加え て さっ と 炒め る 。
さらに 、 ツナ 缶 を 缶汁 ごと 加え 、 パスタ の 茹で汁 大さじ 4 と 小松菜 の 茎 を 加え 、 フライパン を 揺す り ながら 強火 で 加熱 し て から 1 分 ほど し て ソース が とろり と し て き た ら 一 度 火 を 止め る 。
