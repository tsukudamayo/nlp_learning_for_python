日本 酒 と しょう油 で 下味 を つけ る 。
万能 ねぎ （ 白 い 所 ） と 鷹 の 爪 を 刻 ん で 、 ★ の 調味 料 と 混ぜ て タレ を 作 る 。
鶏 もも肉 の 余分 な 水分 を ペーパー タオル で 取 り 、 片栗粉 を しっかり と 押さえ て つけ る 。
鶏 もも肉 に つけ た 片栗 粉 の 余分 な 粉 を 落と し て 170 度 の サラダ 油 で 約 3 分 揚げ る 。
