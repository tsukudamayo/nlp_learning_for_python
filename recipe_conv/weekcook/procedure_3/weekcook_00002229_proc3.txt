生姜 を 薄切り に し て 、 白ねぎ の 青 い 部分 は 小鍋 に 入 る 長 さ に 切 る 。
小鍋 に 、 鶏 もも肉 と 白ねぎ 、 生姜 、 水 （ 450 ml ） 、 日本 酒 、 塩 を 加え て 火 に かけ て 、 アク を 取り ふつふつ する 程度 の 火 加減 で 約 10 分 煮 る 。
10 分 後 火 を 止め 、 白ねぎ と 生姜 を 取り出 し 、 鶏肉 を 煮汁 に 漬け た まま 常温 に な る まで 冷ま す （ 茹で 汁 150 ml は 残 し て お く ） 。
ぶな しめじ は 石づき を と っ て 、 白ねぎ は 厚 さ 1 cm の 斜め切り に する 。
茹で 鶏 は 、 厚 さ 1 ～ 2 cm に 切 る 。
鍋 に 、 ぶな しめじ と 白ねぎ 、 茹で 鶏 の 茹で汁 （ 150 ml ） 、 水 （ 100 ml ） を 加え て 火 に かけ て 、 ふつふつ と する 程度 の 火 加減 で 、 野菜 が 柔らか く な る まで 煮 て 、 ナンプラー を 加え て 味 を 調え て スープ を 作 る 。
スープ に うどん （ 必要 で あ れ ば 袋 の 表示 に したが っ て 茹で て お く ） を 加え て 温め る 。
