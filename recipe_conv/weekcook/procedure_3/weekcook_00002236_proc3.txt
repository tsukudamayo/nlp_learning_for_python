玉ねぎ は 5 mm 程度 の 薄切り に し て 、 舞茸 は 適当 な 大き さ に 裂 き 、 にんにく は 薄切り に する 。
アスパラガス は 、 下 の 方 の 袴 を 取り除 き 、 長 さ を 4 等分 に 切 る 。
豚 ロース （ ステーキ 用 ） は 両面 に 塩 一 つまみと こしょう を 少々 振 っ て 下味 を 付け 、 脂肪 と 赤身 の 間 に 数 カ所 切り込み を 入れ て 肉 の 筋 を 切 っ て お く 。
鍋 に 湯 を 沸騰 さ せ て 1 ～ 2 % の 塩 （ 分量 外 ） を 加え て 、 カット し た アスパラガス を さっと 茹で て 取り出 し 、 冷水 に つけ て 冷ま す 。
フライパン に サラダ 油 大さじ 1 / 4 と にんにく を 入れ て 火 に かけ 、 玉ねぎ を 加え て 炒め て 、 しんなり し て き た ら 舞茸 を 加え て ひと 混ぜ し 、 ケチャップ を 加え て 炒め合わせ て ケチャップ ソース を 作 る 。
フライパン に サラダ 油 を 入れ て 火 に かけ 、 油 が 温ま っ て き た ら 回 し 広げ 、 豚 ロース （ ステーキ 用 ） を 加え て 焼 き 、 横 から 見 て 厚 さ の 半分 程度 まで 色 が 変わ っ たら 裏返 し 、 同様 に 焼 く 。
