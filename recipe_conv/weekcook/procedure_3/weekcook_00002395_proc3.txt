耐熱 容器 に 薄力 粉 と 牛乳 を 入れ て よく 混ぜ る 。 電子 レンジ で 約 2 分 加熱 し 、 バター を 加え て よく 混ぜ る 。 更に 約 2 分 加熱 し 、 塩 を 加え て ホワイト ソース を 作 る 。
じゃがいも は 洗 っ て 、 電子 レンジ で 8 ～ 9 分 加熱 する 。
じゃがいも は 熱 い うち に 皮 を 剥 い て 一口 大 に 切 る 。 玉ねぎ は 薄切り に する 。
白身 魚 ( 切り身 ) は 食べ 易 い 大き さ に 切 り 、 水分 を ペーパー タオル で 拭き取 る 。
フライパン に 白身 魚 と 玉ねぎ を 加え 、 白 ワイン を 振 っ て から 蓋 を し て 蒸し焼き する 。 水分 が 無く な っ た ら じゃがいも と ホワイト ソース を 加え て 混ぜ合わせ 、 とろみ が 付 い たら 火 を 止め る 。
フライパン の 白身 魚 の グラタン を グラタン 皿 に 移し変え 、 チーズ ( ピザ 用 ) を 載せ て 230 度 の オーブン で 約 10 分 、 こんがり 焦げ目 が つ く まで 焼 い たら 完成 。
