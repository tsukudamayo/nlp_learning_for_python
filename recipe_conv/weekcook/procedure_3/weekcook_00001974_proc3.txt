生姜 は 細 い 千切り に する 。 人参 は 細 い 部分 なら 厚 さ 2 ～ 3 mm の 輪切り 、 太 い 部分 なら 半月 切り に する 。
フライパン に サラダ 油 を 入れ て 中火 で 熱 し 、 人参 と 生姜 を 加え て 炒め る 。
しんなり し て き た ら 、 しょう油 と みりん を 加え 、 汁気 が なくな る まで 炒め て 、 人参 と 生姜 の テリヤ キ 炒め を 作 る 。
パプリカ は 5 mm 幅 の 千切り に し て 、 人参 と 生姜 の テリヤ キ 炒め と 混ぜ合わせ る 。
更に マヨネーズ を 加え 、 混ぜ合わせ て から 器 に 盛り付け 、 粗挽き 黒 こしょう を 振 っ て 完成 。
