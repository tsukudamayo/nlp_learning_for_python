鰯 ひらき は 、 水洗い し て から 水気 を 拭き取 り 、 塩 と こしょう で 下味 を 付け る 。
トマト は 厚 さ 2 mm の 薄切り に する 。
バジル （ 葉 ） 、 ピーナッツ は 細か く 刻 む 。
★ の 調味 料 と ピーナッツ を 混ぜ合わせ て お く 。
じゃがいも と 玉ねぎ は 、 皮 を 剥 い て 厚 さ 2 mm の 薄切り に する
カット し た じゃがいも と 玉ねぎ を 深め の 電子 レンジ 可能 な 容器 に 移 し 、 白 ワイン と おろし にんにく を 混ぜ合わせ た ソース を かけ 、 塩 と こしょう を 振 っ て 絡 ま せ た 後 、 ラップ を し て 電子 レンジ （ 600 W ） で 4 分 加熱 する
電子 レンジ 可能 な 容器 の 上 に 加熱 し た じゃがいも と 玉ねぎ を 、 厚み が なるべく 少な く な る よう に 平ら に 広げ 、 その 上 に トマト と バジル を 載せ て 塩 と こしょう を 振り掛け る 。
電子 レンジ 可能 な 容器 の トマト の 上 に 鰯 の 皮目 を 上 に し て 、 鰯 が 重な ら な い よう に 置 く 。
鰯 の 上 から 混ぜ合わせ て お い た ★ の 調味 料 と ピーナッツ を かけ 、 軽 く 手 で 押さえ て から 、 グリル ( トースター で も 可 ) で 8 分 焼 く 。
鰯 と トマト の 重ね 焼き を お 皿 に 盛り付け たら 完成 。
