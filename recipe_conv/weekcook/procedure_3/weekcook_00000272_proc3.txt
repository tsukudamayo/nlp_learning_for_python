レタス は 幅 1 cm に 切 り 、 きゅうり は 細切り に する 。 ★ の 白ねぎ は 刻 み 、 ★ の 鷹 の 爪 は 細か く 刻 ん で 他 の ★ の 調味 料 と 混ぜ合わせ て ねぎ ソース を 作 る 。
鶏 む ね 肉 は 皮 を 取り除 き 、 観音 開き に な る よう に 包丁 を 入れ 厚 さ を 均一 に する 。 塩 と 砂糖 を 揉 み込 み 、 スライス し た 生姜 、 臭み 取 り用 の 白ねぎ （ 青 い 所 ） と 一緒 に 鍋 に 入れ 日本 酒 を 振 る 。
鍋 に 蓋 を し て 沸騰 し たら 弱火 に し て 約 3 分 火 に かけ る 。 鶏 む ね 肉 を 裏返 し て 約 3 分 加熱 する 。
鶏 む ね 肉 は 約 5 分 蒸ら す 。 鶏 む ね 肉 に 火 が 通 っ たら 取り出し 繊維 に 添 っ て 細 く 裂 く 。
