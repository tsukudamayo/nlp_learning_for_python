耐熱 容器 に 干し 椎茸 と ひたひた の 水 を 入れ 、 蓋 なし で 電子 レンジ （ 600 W ） で 約 20 ～ 30 秒 加熱 後 、 石づき を 取 る 。
ごぼう は 皮 を こす り 落と し て から 大きめ の 乱切り に し て 、 むき 栗 は ざっと 水洗い する 。
鶏 もも肉 は 黄色 の 脂肪 を 取り除 い て 一口 大 に 切 り 、 日本 酒 と しょう油 で 下味 を 付け る 。
フライパン を 火 に かけ て サラダ 油 を 加え て から 鶏肉 を 炒め る 。
鶏肉 の 色 が 変わ っ て き たら 、 ごぼう と 椎茸 、 むき 栗 を 入れ て 、 水 と still たれ 、 オイスター ソース を 加え て 煮含め る 。
煮汁 が 無くな っ て き たら 味 を み て 、 薄 い よう なら still たれ を 加え て 調整 し て から 火 を 止め る 。
鶏 と 栗 の 炒め煮 を 器 に 盛り付け 完成 。
