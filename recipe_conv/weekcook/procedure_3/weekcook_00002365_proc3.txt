長 なす は 縦 半分 に 切 っ て から 、 皮 と 実 の 間 に 切り込み を 入れ る 。 生姜 は すりおろ し 、 絞 っ て 生姜 汁 に する 。
長 なす の 表面 に サラダ 油 ( 分量 外 ) を 塗 っ て 、 グリル ( フライパン で も 可 ) で 約 15 分 焼 く 。
小鍋 に 鶏 ミンチ に 酒 小さじ 1 / 2 ( 分量 外 ) を 加え て よく 混ぜ る 。 火 に かけ て 肉 そぼろ に する 。
鍋 に ★ の 調味 料 を 入れ て よく 混ぜ て 火 に かけ 、 煮詰め て とろみ が つ い た ら 、 肉 そぼろ と 生姜 汁 を 加え る 。
なす を 器 に 盛り付け 肉 味噌 を かけ 、 お 好み で 千切り に し た 生姜 を 載せ たら 完成 。
