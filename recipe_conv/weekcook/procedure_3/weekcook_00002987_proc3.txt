干し 椎茸 は 、 耐熱 容器 に 干し 椎茸 と ひたひた の 水 を 入れ 、 蓋 なし で 電子 レンジ （ 600 W ） で 約 20 ～ 30 秒 加熱 し て 戻 す
干し 椎茸 と 人参 と 筍 （ 水煮 ） は 千切り に し 、 絹ごし 豆腐 （ 小 ） は 幅 5 mm の 細切り に する 。
鶏 ささみ は 幅 5 mm の 細切り に し 、 日本 酒 と 片栗粉 を まぶ す 。
鍋 に 水 と 干し 椎茸 と 人参 、 筍 を 加え て 火 に かけ 沸騰 し て き た ら 鶏 ささみ と 豆腐 を 加え る 。 再び 沸騰 し て き た ら 、 鶏がら スープ の 素 と 薄口 しょう油 、 こしょう で 味 を 調え て 、 水溶き 片栗粉 で とろみ を つけ る 。
とろみ が つ い た ところ へ 溶き卵 を 少し ずつ 流し入れ て 蓋 を し て 、 火 を 止め て 蒸ら す 。
