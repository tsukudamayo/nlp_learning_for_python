ナス は ヘタ を 取 っ て から 縦 に 4 等分 に 切 り 、 塩水 ( 分量 外 ) に 浸け て アク を 抜 い た 後 、 水分 を ペーパー タオル で 拭き取 る 。
深め の 耐熱 容器 に ナス を 入れ 、 サラダ 油 を 注 い で ナス に 絡め た 後 、 ラップ を し て レンジ で 約 8 分 加熱 する 。
★ の 生姜 は すりおろ し 、 他 の ★ の 調味 料 と 混ぜ合わせ て 生姜 ソース を 作 る 。
大葉 は 千切り に する 。
加熱 し た ナス を お 皿 に 盛り付け 、 イカ ( 刺身 用 ) を ナス の 上 に 載せ て 生姜 ソース を かけ 、 大葉 を 散ら し たら 完成 。
