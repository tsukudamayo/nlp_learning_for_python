鯵 （ 二 枚 おろし の 物 ） は 水分 を 拭き取 り 、 塩 を 振 っ て 10 分 寝かせ る 。
玉ねぎ 、 セロリ は 薄切り 、 人参 は 千切り に する 。
玉ねぎ は 水 に さら し て から 水気 を ペーパー タオル で ふき取 る 。
鯵 から 出 て き た 水分 を ペーパー タオル で 拭き取 り 、 塩 と こしょう を 振 っ て から 片栗粉 を 全体 に つけ る 。
180 ℃ の サラダ 油 で からり と 揚げ る 。
鷹 の 爪 は 小口 切り に し て ★ の 調味 料 と 一緒 に 混ぜ合わせ て から 小鍋 に 移 し て 火 に かけ 、 一 煮立ち さ せ て タレ を 作 る 。
蓋付 の 容器 に 野菜 と 揚げ た 鯵 を 入れ て 、 上 から 作 っ た タレ を 注ぎ入れ 、 タレ に 浸け る 。
